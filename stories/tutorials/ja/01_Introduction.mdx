# GAKUYA 使い方ガイド

GAKUYAへようこそ！このガイドでは、GAKUYAを使ったUTAU音源のパッケージング方法を順を追って説明します。

## GAKUYAとは

GAKUYAは、原音設定が完了したUTAU音源を配布できる状態にするためのWebアプリケーションです。
インストール不要で、ブラウザ上でUTAU音源の配布準備に必要な作業を行えます。

> 💡 **UTAUや音源配布について詳しく知りたい方は**、[UTAUとは](/docs/tutorial-ja-02-utauとは--docs)をご覧ください。

## できること

- 📝 **character.txt の作成・編集** - 音源情報（名前、作者、アイコンなど）を設定
- 📄 **readme.txt の作成・編集** - 利用規約や音源説明を記載
- 📦 **install.txt の作成・編集** - UTAU向けインストールzipの設定
- 🎵 **character.yaml の作成・編集** - OpenUTAU向け設定ファイルの一部対応
- 🎹 **prefix.map の作成・編集** - 多音階音源用の音高別設定
- 🗑️ **不要ファイルの削除** - キャッシュファイルなど配布不要なファイルを除去
- 📊 **周波数表ファイル（frq）の生成** - UTAU標準の周波数表を自動生成
- 🔊 **wavファイルの形式変更** - モノラル化、サンプリングレート調整、DCオフセット除去など

## できないこと

GAKUYAでは以下の作業はできません。別のツールやソフトウェアを使用してください。

- 🎤 **録音** - 音声の録音は専用の録音ソフトを使用
- ✂️ **原音設定** - oto.iniの編集には[LABERU](https://k-uta.jp/laberu/)などの使用を推奨
- 🖼️ **画像の準備** - アイコンや立ち絵は事前に用意が必要（登録はGAKUYAで可能）
- 🔧 **破損した周波数表の修正** - 周波数表の修正には専用ツールが必要

## 必要なもの

- モダンなWebブラウザ（Chrome、Firefox、Safari、Edgeなど）
- 原音設定が完了したUTAU音源（wavファイルとoto.iniが含まれたzip）
- （オプション）アイコン画像や立ち絵画像
