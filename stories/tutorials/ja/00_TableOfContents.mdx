## 学習の進め方

### 初めての方（最短ルート）

以下のページだけで配布可能な音源が完成します（約10～20分）：

**01 → 02 → 03 → 04 → 05 → 06 → 07 → 11**

### より良い音源を作りたい方

基本ルートに加えて、以下も設定しましょう：

- **08.利用規約を書く** - 配布音源には必須です
- **10.多音階設定** - 音域の広い音源を作る場合に便利

### 全部設定したい方

01から11まで順番に進めてください。
08（利用規約）、09（インストール設定）、10（多音階設定）は、必要に応じてスキップしても問題ありません。

### スマートフォンで作業したい方

GAKUYAはスマートフォンでも使えます。基本ルートを順番に進めてください。
画像の準備は、スマホの画像編集アプリで作ったPNG画像が使えます。
